## <span id="">二、CSS3背景</span>

三个主要属性：

+ background-size
+ background-origin
+ background-clip


### background-size

```css
#example1 {
    border: 1px solid black;
    background:url(img_flwr.gif);
    background-repeat: no-repeat;
    padding:15px;
}

```
